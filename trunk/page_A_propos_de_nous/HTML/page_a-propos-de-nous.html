À Propos de Nous

Mission du projet Accidentologie

Le projet Accidentologie vise à créer un site web regroupant tous les accidents de la route en France. Les conditions pour que ce site soit utilisable sont que les accidents puissent apparaître sur une carte et que l'utilisateur puisse filtrer les accidents en fonction de critères prédéterminés.
Ce site a également une visée pédagogique, en montrant quels sont les facteurs à risque sur la route.

Description du site web

Ce site comprend deux pages principales:
Une page carte, sur laquelle on affiche tous les accidents présents dans l'API. Sur cette page, l'utilisateur peut filtrer les accidents sur un certain nombre de critères, notamment le lieu, la date ou encore les conditions météorologiques.
Une page statistiques, sur laquelle l'utilisateur peut afficher un graphique représentant le nombre d'accident en fonction d'une donnée de son choix (le lieu, la date,...). Chaque donnée a un affichage préprogrammé, pour une lisibilité optimale de chaque donnée.

Les projet futurs

Ce site pourra connaître de nombreuses améliorations.
Premièrement, il est extensible à n'importe quelle région du monde et il est envisageable de l'adapter pour d'autre pays.
Ensuite, il est possible d'améliorer la page statistiques, avec un tableau plus performant par exemple.
Enfin, on peut imaginer ajouter de nombreux filtres afin d'aider l'utilisateur à affiner au mieux sa recherche.
